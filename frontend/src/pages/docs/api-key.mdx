# API Key

## Generate API key

To generate or change API key for your project, go to the Settings tab inside your project an click on the `Change Key` button

<Image src="/images/api-key.png" alt="Hello" width={500} height={500} />

## Remove API key
If you want to remove your API key, inside the `Settings` page click on `Remove Key` button
