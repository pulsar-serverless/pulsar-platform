# Source code

## Download source code

With pulsar you can download a starter code to start fresh, or you can download the latest source code of your serverless app.

to download your project's source code, inside your project go to the `Deployment` tab. and click on the `Download` button

<Image src="/images/deployment-tab.png" alt="Hello" width={500} height={500} />

## Upload source code

With pulsar you can upload your zipped serverless application code 
inside the `Deployment` tab. and click on the `Select file` button and upload your file
<Callout>
Make sure that the code, including the package.json file, is located in the root directory.
</Callout>
