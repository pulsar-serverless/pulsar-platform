## Create Project
<Steps>
### Step 1

To create your new Project on pulsar, first navigate to the `Projects` tab
<Image src="/images/projects-link.png" alt="Hello" width={500} height={500} />

### Step 2

Next the projects page appears, Then click on the `Create project` button.
<Image src="/images/create-project.png" alt="Hello" width={500} height={500} />

### Step 3

Then you will be presented with a create project form. Enter the name and create the project
<Callout>
You can only enter alpha-numeric characters or -
</Callout>

<Image src="/images/create-project-form.png" alt="Hello" width={500} height={500} />

### Step 4

Congratulations! you have successfully created your first project on pulsar. you can now explore more about the project. 
<Image src="/images/project-created.png" alt="Hello" width={500} height={500} />

</Steps>
